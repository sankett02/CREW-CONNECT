# Screen List & Navigation Flow

## 1. Public / Auth
- **Landing Page**: Value prop, login/signup CTA
- **Sign Up / Login**: Role selection (Brand/Creator/Writer/Editor) -> Form -> Onboarding (Profile setup)

## 2. Dashboard (Role-based)
- **Brand Dashboard**: Active projects, Pending applications, CTA "Post New Project"
- **Creator Dashboard**: Recommended open projects, Active projects, Invites
- **Writer/Editor Dashboard**: Active projects, Invites

## 3. Profiles
- **View Profile**: Shows verification badge, niche tags, skills, portfolio links, average rating.
- **Edit Profile**: Form to update info.

## 4. Project Discovery & Creation
- **Post Project (Brand)**: Form (Brief, Niche, Budget, Deadline)
- **Project List (Creators)**: Search/Filter by niche.
- **Project Details**: View brief, CTA "Apply".

## 5. Project Workspace (The Core Wedge)
Navigation Tabs within Workspace:
- **Overview**: Brief, Budget status (Manual Escrow record).
- **Team**: List of members. CTA "Add Member" for Lead Creator.
- **Milestones**: Kanban or List view (Script -> Edit -> Final). Submissions & Approvals.
- **Chat**: Real-time or chronological message thread for the team.

## 6. Post-Project
- **Rating Modal**: Appears when project marks COMPLETED.
- **Dispute/Flagging Modal**: Accessible from Workspace or Profile.

## 7. Admin Panel (Hidden from regular users)
- **User Management**: List users, toggle verify badge.
- **Dispute Log**: List of reported issues.
